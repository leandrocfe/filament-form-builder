## Installation

```bash
git clone https://github.com/leandrocfe/filament-form-builder.git
```

Run the commands:

```bash
cd filament-form-builder
cp .env.example .env
composer install
```

Laravel sail:

```bash
sail up -d
```

Run the commands:

```bash
sail npm i
sail npm run dev
```
